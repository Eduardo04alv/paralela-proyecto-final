# Proyecto Final - Programación Paralela

## Descripción
Este proyecto tiene como objetivo demostrar el uso de programación paralela mediante descomposición exploratoria.

## Temas
1. Generador procedural de islas en matrices
2. Búsqueda de rutas en grafos (exploración paralela)

## Estructura del proyecto
- /docs: Documentación
- /src: Código fuente
- /tests: Pruebas
- /metrics: Resultados de rendimiento
